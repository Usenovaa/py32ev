Линукс команды

1. pwd (print working directory) -> возвращает в терминал абсолютный путь 
до папки (показывает где мы находимся)
2. ls (list) -> выводит содержимое директории (список файлов и папок текущей
директории)
	2.1 ls -a -> выводит все файлы и деректории, включая скрытые
	2.2 ls -l -> выводит все содержимое (кроме скрытых) с дополнительной информацией
	2.3 ls -la -> выводит все содержимое, включая скрытые директории и
 файлы с дополнительной инфо
3. cd  /  cd ~  (change directory) -> переход в домашнюю директорию
	3.1 cd <dir_name> -> переход в директорию dir_name
	3.2 cd /home/user/dir/dir -> переход в директорию по абсолютному пути
	3.3 cd .. -> подняться на уровень выше (вернуться к предыдущей
 директории)
4. mkdir (make directory) -> создает директорию  
	mkdir <name> 
5. touch <filename> -> создает указанный файл
6. rmdir (remove directory) -> удаляет пустую директорю 
	rmdir <dir_name>
7. rm -rf (remove recursive force) -> принудительное удаление, удаляет
 директории с содержимым
8. rm (remove) -> удаляет файл
	rm <filename>
9. clear -> очищает терминал
10. cp (copy) -> копирует файл или директорию в другую директорию
	cp <что копируем filename> <куда копируем dir_name>
11. mv (move) -> 1. перемещает файл
	mv <filename> <dir_name> 
    mv -> переименовывает
	mv <filename> <new_filename>
12. nano -> создает и/открывает файл в встроенном текстовом редакторе
13. echo -> записывает содержимое в файл 
	echo текст > filename
14. cat -> выводит содержимое файла в терминал
	cat <filename>
    cat -> записывает содержимое файла в другой файл
	cat filename > filename2
15. history -> выводит испорию команд
16. du (disk usage) -> выводт исформацию о занимаемой памяти 
17. find -> ищет указанный файл или директорию
	file /home -name <filename/dir_name> -> выводит абсолютный путь 
18. sudo -> утилита, предоставляет права администратора (выполняет команду
 с правами админа)
19. code . -> открывает текущую директорию в vscode

Нажатие tab -> приводит к автозаполнению
стрелками вверх/вниз -> можно перемещаться по использованным командам
чтобы закрыть текстовый редактор nano -> ctrl + S >> ctrl + X
					ctrl + X >> Y >> Enter 
ctrl+alt+T -> открывает терминал
